## 기능 설계

### A. 수업 후 평가

- **트리거**: ClassBoard 퇴장 → 예약 탭 진입 시 미평가 수업 감지
- **노출**: 하루 1회, 가장 최근 수업 1건
- **플로우**:
  - 1~5점 별점 선택
  - 긍정(3~5점): 좋았던 점 복수 선택 → 제출
  - 부정(1~2점): 아쉬운 점 복수 선택 + 차단 제안 체크박스 → 제출
- **저장 규칙**:
  - "다음" 클릭 → 별점만 저장
  - "제출" 클릭 → 별점 + 사유 저장
  - "건너뛰기" 또는 X 닫기 → 저장 안 함
- **건너뛰기**: 클릭 가능한 텍스트 (CTA 버튼 아님)

### B. 튜터 차단

- **진입점 3가지**:
  1. 수업 후 평가에서 부정(1~2점) 시 차단 제안
  2. 수업 이력 상세 페이지에서 직접 차단
  3. 차단 관리 페이지에서 추가
- **한도**: 언어당 5명 (영어/일본어 독립)
- **한도 초과**: 안내 메시지 + 차단 관리 페이지 이동 유도

### C. 차단 관리 페이지

- **위치**: MyPage > 수업 설정 > 튜터 관리
- **기능**: 차단 목록 조회, 해제, 비활성 튜터 라벨 표시
- **표시 항목**: 튜터 프로필, 차단 일시, 마지막 수업 일시
